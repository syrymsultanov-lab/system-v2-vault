# SKILL: Generator — Генерация контента

## Назначение
Этот skill описывает, как система генерирует текст подписи, хэштеги и атрибуцию для каждого найденного материала.

---

## Входные данные

```json
{
  "title": "Luxury Cruise Ship Tour — MSC Bellissima",
  "author_name": "Cruise With Ben And David",
  "author_url": "https://youtube.com/channel/UCxxx",
  "source_url": "https://youtube.com/watch?v=abc123",
  "thumbnail_url": "https://...",
  "platform": "instagram"
}
```

---

## Промпт для генерации подписи (Claude API)

### Системный промпт
```
Ты — контент-менеджер бренда Сайры Султановой (sairateam.com), дистрибьютора InCruises.
Твоя задача — писать вдохновляющие посты о путешествиях, круизах и отдыхе на русском языке.

Тон: тёплый, как рекомендация близкой подруги. Никакой рекламной навязчивости.
Стиль: живой, личный, с эмодзи. Не формальный.
Всегда заканчивай призывом к действию или вопросом к аудитории.
```

### Пользовательский промпт (шаблон)
```
Напиши подпись к посту для {platform} на основе этого видео:

Заголовок видео: {title}
Автор: {author_name}

Требования:
- Длина: {caption_length} (instagram: 150-300 символов, youtube: 500-800 символов)
- Начни с цепляющего первого предложения (без приветствий типа "Привет!")
- Упомяни тему путешествия или круиза
- Добавь личное отношение от имени Сайры
- Заверши вопросом к аудитории ИЛИ призывом перейти по ссылке в bio
- НЕ упоминай конкурентов InCruises
- Не используй слова: "потрясающий", "невероятный", "уникальный" (заезженные)

Верни ТОЛЬКО текст подписи, без объяснений.
```

---

## Генерация хэштегов

### Промпт для хэштегов
```
Сгенерируй 20 хэштегов для поста о: {title}
Платформа: {platform}

Правила:
- 10 на русском языке: #круиз #путешествие #отдых и т.д.
- 10 на английском: #cruise #travel #luxury и т.д.
- Включи: #sairateam #incruises #сайра
- Только релевантные теме, без спамных тегов
- Формат: через пробел, начиная с #

Верни ТОЛЬКО строку хэштегов, без объяснений.
```

---

## Формирование атрибуции

### Шаблон (обязательный блок в каждом посте)

```
📹 Видео: {author_name}
🔗 Оригинал: {source_url}
```

**Для Instagram** (ссылки не кликабельны):
```
📹 Видео: @{instagram_handle если известен, иначе author_name}
▶️ Смотреть оригинал — ссылка в bio
```

**Для YouTube Community Post**:
```
📹 Видео: {author_name} → {source_url}
```

## Итоговая структура поста

### Instagram
```
{caption}

{hashtags}

—
📹 Видео: {author_name}
```

### YouTube Community Post
```
{caption}

📹 Видео: {author_name}
🔗 {source_url}

{hashtags}
```

```json
}
```

---

## Контроль качества

После генерации проверить:
1. Длина подписи соответствует платформе
2. Присутствует блок атрибуции
3. Присутствуют хэштеги
4. Нет запрещённых слов (список в `config/platforms.json`)
5. Язык — русский (основной текст)

Если проверка не прошла → регенерировать (не более 2 попыток).
