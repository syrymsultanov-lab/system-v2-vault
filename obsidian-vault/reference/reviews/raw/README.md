# Reviews — RAW архив

Назначение: оригиналы отзывов от Сайры/партнёров **без обработки**.

## Что сюда кидать
- Фото скринов TG/WhatsApp/Instagram
- Voice messages (.m4a, .ogg, .mp3)
- Видео-отзывы (.mp4, .mov)
- Текст в свободной форме (.md, .txt)

## Формат имени файла
`YYYY-MM-DD_partner-name_kind.ext`

Примеры:
- `2026-05-23_anna-k_screenshot.jpg`
- `2026-05-23_anna-k_voice.m4a`
- `2026-05-23_anna-k_text.md`

## Что НЕ делать
- Не редактировать оригинал
- Не перемещать сюда обработанные версии (для них есть `docs/reviews/` и `assets/images/reviews/`)
- Не коммитить большие медиа (исключены через `.gitignore`)

## Пайплайн
RAW (здесь) → ручная обработка Сырыма → `docs/Reviews.md` (KB) + `assets/images/reviews/` (лендинг)
