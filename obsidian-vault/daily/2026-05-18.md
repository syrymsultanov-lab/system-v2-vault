## Решения

- **KB не цензурим, compliance через hedge в промпте**. Сырым: «если из официального источника — не удалять». Hedge-фраза дословно из официальной Comp Plan slide («Это не обещание дохода, а демонстрация»). Применяется ко всем числам бонусов/дохода/процентов/ранговых сумм. Стоимость пакетов и активации — расходы, не требуют hedge. См. [[Business Rules]] §5 + memory `feedback_compliance_income_hedge`
- **TG identifier = `tg_chat_id` (numeric)**, не username/handle. Стабильный, всегда есть после `/start <uuid>`. Все будущие inbound/outbound nodes ОБЯЗАНЫ использовать его. См. memory `reference_tg_chat_id_stable_identifier`
- **STOP intent — обязательное Meta/WABA правило**. 13 синонимов case-insensitive, hard-stop без анализа KB
- **Media → автоматическая эскалация**. Никогда не звать LLM на media payload, всегда передавать партнёру с текстом «Секунду, передаю Сайре»
