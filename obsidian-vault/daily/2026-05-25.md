### 1. Диагностика — почему AI не пишет form-лидам

Сырым спросил: 3 кандидата с лендинга (Салтанат @Saltavipstar, Батима, Салтанат @Saltavip) от 2026-05-23 не получили сообщений от AI.

**Root cause**: TG Bot API не может писать первым по `@handle` / phone. Нужен numeric `chat_id`, который появляется только когда юзер сам напишет боту (или нажмёт deep-link `t.me/bot?start=X`).

Pipeline дошёл до WF9 dispatch и встал:
```
landing form → leads → trigger create_contact_from_lead →
  contact (ai_consent=true, tg_chat_id=NULL) →
  Conversation Loop → outbound status='dry_run' →
  WF9 → ❌ нет tg_chat_id → молчание
```

3 form-лида сидят с `dry_run` greeting'ами. AI proactive greeting **не дойдёт без deep-link** (вариант C из SYSTEM V2.md:157).
