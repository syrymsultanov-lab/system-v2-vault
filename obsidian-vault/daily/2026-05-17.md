## Решения

- **Budget defaults**: `AI_BUDGET_DAILY_USD=2`, `AI_BUDGET_MONTHLY_USD=50` через `Number($env.X || N)` fallback в Code-ноде Watchdog. Явные env vars опциональны (если нужны другие значения — добавить в VPS `.env` + `docker-compose.yml environment:` + `compose up -d`)
- **Cost calc inline**: hardcoded $0.15/M + $0.60/M в `85_Log_AI_Job_Run`. При смене модели обновлять inline (не хочу отдельную таблицу model_rates пока)
- **Pricing tier**: $0.0000015 per input token, $0.0000006 per output token (gpt-4.1-mini Jan 2026 rates)
- **`ai_config` JSONB vs structured columns**: оба пути есть. Typed = source of truth, JSONB = bucket для future без миграций
- **Followup limit**: 3 нудж, потом `cold`. Auto-stop. AI решает текст нуджа, не интервал (интервал зашит в RPC)
- **WF leaves**: `91_Notify_TG` / `95_Log_Failure` в Conversation Loop — terminal ноды by design. `continueOnFail=true`. Bug нет
