# Reviews — социальное подтверждение для AI

Эталонные отзывы партнёров команды Сайры. Источник: **официальный журнал компании inCruises** (consent через публикацию в корпоративном источнике). AI цитирует в ответах на возражения «работает ли это для обычных людей», «есть ли реальные результаты», «кто из команды добился успеха».

## Compliance

- Ст.17(в) 104RU: никаких цифр дохода «X заработал $Y за Z дней». Допустимы качественные результаты: круизы, страны, ранги, размер команды, исполнение мечт
- Ст.17(г): не приводить личные доходы конкретных людей даже на их же примерах
- Каждая история взята из публикации в журнале inCruises = consent на публичное использование
- Disclaimer обязателен в карточках и в AI-ответах: «результаты индивидуальны, доход не гарантирован»

## Как использовать в AI

При вопросе клиента «есть ли реальные люди в команде Сайры?», «кто из обычных мам/банкиров добился?», «какие результаты?» — AI цитирует 1-2 кейса из ниже + обязательный hedge.

**Допустимые паттерны цитирования:**
- «В команде Сайры [Имя], [ранг]. [одна фраза истории]. Результаты индивидуальны.»
- «Например, [Имя] из [город] [качественное достижение]. У каждого свой путь.»

**Запрещено:**
- Цитировать ранг как обещание дохода
- Сравнивать «Лаура за N месяцев — а ты можешь так же»
- Обещать что новый партнёр повторит результат
